选择器的作用

CSS主要的作用就是给网页中的dom元素设置样式，选择器则是用来匹配dom元素的。

选择器分为哪几类呢？

相信大家都会使用选择器了，那么到底有哪些选择器呢？现在让我们具体来学习和巩固一下吧！

通配符

*

可以选择所有元素

ID选择器

#firstname

选择 id="firstname" 的所有元素。

类选择器

.intro

选择 class="intro" 的所有元素。

元素选择器

p

选择所有的标签

后代选择器

h1 em {color:red;} 对 h1 元素中的 em 元素应用样式

子元素选择器

只能选择作为某元素子元素的元素。 写法：h1 > strong {color:red;} 解释为“选择作为 h1 元素子元素的所有 strong 元素”。

选择器分组

h1, h2, h3, h4, h5, h6 {color:blue;}

以上所有元素的背景都会变成蓝色

选择相邻兄弟 h1 + p {margin-top:50px;}

如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器

伪类

a:link/visited/hover/active {color: #FF0000}

:first-child 伪类

div p:first-child {font-weight: bold;} 指div下面的第一个子元素p

伪元素

CSS 伪元素用于向某些选择器设置特殊效果。

:first-line 伪元素

p:first-line{color:#ff0000;font-variant:small-caps;} "first-line" 伪元素用于向文本的首行设置特殊样式。

:first-letter 伪元素

"first-letter" 伪元素用于向文本的首字母设置特殊样式：

CSS2 - :before 伪元素

":before" 伪元素可以在元素的内容前面插入新内容。

CSS2 - :after 伪元素

"":after" 伪元素可以在元素的内容之后插入新内容。

属性选择器

a[href] {color:red;}} 可以只对有 href 属性的锚（a 元素）应用样式

a[href][title] {color:red;} 将同时有 href 和 title 属性的 HTML 超链接的文本设置为红色

具体属性值选择器

可以进一步缩小选择范围，只选择有特定属性值的元素。

a[href="http://www.w3school.com"] {color: red;} 将指向 Web 服务器上某个指定文档的超链接变成红色

a[href="#"][title="W3School"] {color: red;} 多个属性-值选择器链接在一起来选择一个文档。

planet[moons="1"] {color: red;} 属性与属性值必须完全匹配

[class~="important"] {color: red;} 根据属性值中的词列表的某个词进行选择，
